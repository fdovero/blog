Mon Firefox

dim. 17 novembre 2013
By Fabien Dovero

Je suis un amoureux du Web.

De tous les navigateurs, Firefox à toujours eu ma préférence depuis la version 2.0 qu’un ami m’avait installé pour remplacer IE6… Ce fut le coup de foudre ! Je n’ai plus jamais quitté mon panda roux !

Je tiens à jour cet article pour m’aider à maintenir mes differentes installations.

Les extensions

Je vais lister ici mes extensions. En règle générale j’essai de ne pas trop en installer pour ne pas allourdir le chargement et de préférence celles qui peuvent travailler sans laisser de trace dans le DOM ( j’ai par exemple désinstaller turn off the light à cause de sa balise script qu’il insère en entete de chaque page web ).

  • Adblock Edge ( avec les listes FR + EasyList, EasyPrivacy, Fanboy’s Social Blocking List, Malware Domains )
  • Adblock Plus Pop-up Addon
  • Disconnect
  • HTTPS-Everywhere
  • Long URL Please
  • Pure URL
  • Redirect Cleaner
  • Smart Referer
  • Self-Destructing Cookies
  • Wot
  • Pentadactyl
  • Click to Play per-element
  • eCleaner
  • DuckDuckGo

Les tweaks

Voici une liste des modifications que j’ai apporté dans le about:config

  • layers.acceleration.force-enabled=true ( pour forcer l’accélération matérielle )
  • network.proxy.socks_remote_dns=true ( pour que les requêtes DNS passent par le proxy socks )
  • layout.spellcheckDefault=4 ( changer le style d’affichage du correcteur d’orthographe )
  • media.gstreamer.enabled=true ( utilise gstreamer pour la balise video - bêta )
  • plugin.sessionPermissionNow.intervalInMinutes=0 ( avec l’addon Click to Play per-element )
  • browser.urlbar.trimURLs=false ( ré-affiche le protocole dans la barre de navigation )
